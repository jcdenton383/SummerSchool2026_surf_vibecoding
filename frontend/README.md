# Frontend MVP "Шеф-стол"

Клиентское мобильное приложение на Expo + React Native + TypeScript.

## Запуск

```bash
npm install
npm run start
```

Переменные окружения:

- `EXPO_PUBLIC_API_BASE_URL=http://localhost:8080` - URL mock-backend для web/эмулятора.
- Для Android-устройства указывайте IP компьютера в локальной сети, например `EXPO_PUBLIC_API_BASE_URL=http://10.129.86.167:8080`.

Локальный mock-слой во фронтенде отключён: приложение получает данные только через HTTP API.

## Реализованный MVP-скоуп

- Авторизация по телефону в demo-режиме.
- Расписание на ближайшие 7 дней и фильтр дат.
- Карточка класса.
- Оформление брони с участниками, аллергиями, инвентарём и offline-payment confirmation.
- Заглушка недоступной онлайн-оплаты.
- Мои брони, детали, добавление участников, отмена брони/участника.
- In-app уведомления.
- Оценка шефа для завершённой брони.
- Legal screen.

Онлайн-оплата, админка, интерфейс шефа, `/studio-info`, `StudioInfo` и `studios` не реализованы.
