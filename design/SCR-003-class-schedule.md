# SCR-003. Расписание классов

## Метаданные

| Поле | Значение |
|---|---|
| Раздел приложения | Расписание |
| Приоритет | Высокий |
| Что реализует | Просмотр доступных кулинарных классов на ближайшие 7 дней и за выбранный период |
| Зависит от экранов | SCR-002 Код подтверждения |
| Следующие экраны | SCR-004 Фильтр даты; SCR-005 Карточка класса; SCR-010 Аккаунт / мои записи |

## Что пользователь должен понять и сделать

Пользователь должен быстро понять, какие классы доступны, когда они проходят, есть ли свободные места и сколько стоит участие. Основное действие - выбрать подходящий класс или изменить диапазон дат.

## Какой контент показать и что в нём главное

Главное - список классов и доступность мест. Экран должен быть удобен для сканирования на мобильном устройстве.

Показать:
- баннер студии со статическим UI/UX-текстом названия и адреса;
- выбранный период, по умолчанию ближайшие 7 дней;
- действие выбора дат;
- карточки классов: название, дата, время, длительность, направление, сложность, шеф, рейтинг, свободные места, цена;
- вход в аккаунт / мои записи.

Не показывать поиск, фильтры по кухне, шефу, цене или сложности в MVP.

## Какие состояния нарисовать

- Загрузка расписания.
- Список доступных классов.
- Пустое состояние «Пока нет доступных классов».
- Ошибка загрузки или плохое соединение.
- Класс без свободных мест.
- Класс отменён студией.
- Обновление списка после смены дат.
