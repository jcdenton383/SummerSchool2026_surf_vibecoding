# SCR-014. Отмена класса студией

## Метаданные

| Поле | Значение |
|---|---|
| Раздел приложения | Аккаунт / Уведомления |
| Приоритет | Высокий |
| Что реализует | Сообщение о форс-мажорной отмене класса студией и инструкцию по возврату |
| Зависит от экранов | SCR-010 Аккаунт / мои записи; SCR-011 Детали брони; push/SMS-уведомление |
| Следующие экраны | SCR-011 Детали брони; SCR-003 Расписание классов |

## Что пользователь должен понять и сделать

Пользователь должен сразу понять, что класс отменён студией, приходить не нужно, повторная запись на этот слот недоступна, а дальнейшие действия связаны с возвратом.

## Какой контент показать и что в нём главное

Главное - статус отмены и инструкция по возврату.

Показать:
- заметный статус «Класс отменён студией»;
- название класса;
- дату и время отменённого класса;
- статический UI/UX-текст адреса студии как визуальный контекст, если он предусмотрен макетом;
- инструкцию по возврату денег;
- пояснение, что повторная запись на этот слот невозможна;
- действия «К деталям брони» и «К расписанию».

## Какие состояния нарисовать

- Открытие экрана из push/SMS.
- Отменённая запись в деталях брони.
- Отменённый класс в списке записей.
- Запрет записи на отменённый слот в карточке класса.
