# Версия 1.1 — фото блюда в карточках классов

## Тип

Feature / Non-MVP.

## MVP-статус

Фото блюда не входит в MVP. Изменение разрешено как расширение версии 1.1 и не должно блокировать базовые сценарии записи.

## Затронутые экраны

- `SCR-003-class-schedule`
- `SCR-005-class-details`

## Затронутые данные

- `ClassProgram.dishImageUrl`
- `ScheduleSlot.program.dishImageUrl`

## UI-правила

- В расписании фото показывается в верхней части карточки класса.
- В расширенной карточке фото показывается перед названием класса.
- При отсутствии URL карточка отображается без изображения.
- Ошибка загрузки изображения не должна блокировать запись.

## API-правило

Поле `dishImageUrl` возвращается в объекте `ClassProgram`.

## Не входит

- Загрузка фото из мобильного приложения.
- Редактирование фото.
- Хранение бинарных файлов.
- Проверка авторских прав на изображения.
