# SCR-010. Аккаунт / мои записи

## Метаданные

| Поле | Значение |
|---|---|
| Раздел приложения | Аккаунт |
| Приоритет | Высокий |
| Что реализует | Просмотр предстоящих и завершённых записей клиента |
| Зависит от экранов | SCR-002 Код подтверждения; SCR-009 Успешная запись |
| Следующие экраны | SCR-003 Расписание классов; SCR-011 Детали брони; SCR-014 Отмена класса студией; SCR-015 Оценка шефа |

## Что пользователь должен понять и сделать

Пользователь должен понять, какие записи у него есть, какие из них предстоящие, завершённые или отменённые. Основное действие - открыть нужную запись и управлять ей.

## Какой контент показать и что в нём главное

Главное - список записей и их статусы.

Показать:
- минимальные данные клиента;
- вкладки или секции «Предстоящие» и «Завершённые»;
- карточки записей с названием класса, датой, временем, статусом, количеством участников;
- заметную метку для отмены студией;
- действие оценки для завершённого класса без оценки;
- переход к расписанию.

## Какие состояния нарисовать

- Есть предстоящие записи.
- Есть завершённые классы.
- Нет записей.
- Есть отменённый студией класс.
- Завершённый класс ожидает оценки.
- Ошибка загрузки списка.
