# Версия 1.1 — календарный выбор дат в расписании

## Тип

Feature / UX refinement.

## Затронутые экраны

- `SCR-003-class-schedule`
- `SCR-004-date-filter`

## Описание изменения

Фильтр дат в расписании должен использовать визуальный выбор даты, а не ручной ввод строки.

## UI-правила

- Пользователь выбирает дату начала и дату окончания через календарный контрол.
- Значения отправляются в API в формате `YYYY-MM-DD`.
- Кнопка сброса фильтра сохраняется.
- Если дата окончания раньше даты начала, приложение показывает ошибку.

## API

API не меняется:

- `GET /slots?from={date}&to={date}`

## Ограничения MVP

- Не добавлять фильтры по шефу, цене, сложности или направлению без отдельного решения.
- Не расширять период по умолчанию: без фильтра backend возвращает ближайшие 7 дней.
