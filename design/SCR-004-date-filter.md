# SCR-004. Фильтр даты

## Метаданные

| Поле | Значение |
|---|---|
| Раздел приложения | Расписание |
| Приоритет | Высокий |
| Что реализует | Выбор периода расписания за пределами ближайших 7 дней |
| Зависит от экранов | SCR-003 Расписание классов |
| Следующие экраны | SCR-003 Расписание классов |

## Что пользователь должен понять и сделать

Пользователь должен понять, какой период расписания сейчас выбирает, и применить его к списку классов. Дополнительная задача - быстро вернуться к периоду по умолчанию.

## Какой контент показать и что в нём главное

Главное - понятный выбор начальной и конечной даты.

Показать:
- текущий выбранный период;
- календарь или два поля «с» и «по»;
- кнопку «Применить»;
- действие «Сбросить к ближайшим 7 дням»;
- кнопку закрытия без применения.

Не добавлять другие фильтры в этот экран.

## Какие состояния нарисовать

- Период по умолчанию.
- Пользователь выбрал начальную дату, но ещё не выбрал конечную.
- Некорректный период.
- Применение выбранного периода.
- Сброс периода.
