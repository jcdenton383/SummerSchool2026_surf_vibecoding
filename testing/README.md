# Testing artifacts

Папка содержит тестовые артефакты для учебного MVP мобильного приложения кулинарной студии "Шеф-стол".

## Состав

- `surface-test-table.md` — таблица поверхностных smoke/regression тестов по известным требованиям, API, frontend и mock-backend.
- `layout-test-table.md` — отдельная таблица тестов layout-компоновки приложения.
- `manual-test-table.md` — таблица ручного тестирования с шагами, тестовыми данными, фактическим результатом, статусом и ссылкой на дефект.

## Ограничение скоупа

Это не полный test plan и не замена unit/integration/e2e тестам. Таблицы фиксируют минимальный поверхностный набор проверок, который нужен для защиты MVP и ручной/полуавтоматизированной регрессии.

Проверки мобильного UI должны выполняться минимум на двух платформах:

- Android: эмулятор или физическое устройство.
- iOS: симулятор или физическое устройство.

Проверка только через Expo Web не считается достаточной для приемки мобильного MVP.
