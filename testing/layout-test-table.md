# Таблица layout-тестов мобильного приложения

## Цель

Проверить поверхностную устойчивость компоновки Expo/React Native приложения на Android и iOS. Эти тесты выделены отдельно, потому что успешная бизнес-логика не гарантирует, что экран usable на разных размерах, с разными safe-area, клавиатурами и системными настройками шрифта.

## Минимальная матрица устройств

| Платформа | Минимум | Дополнительно при наличии времени |
|---|---|---|
| Android | компактный телефон 360x640 или близкий, современный телефон 390x844 | крупный экран/планшет, Android с увеличенным системным шрифтом |
| iOS | iPhone SE/малый экран, iPhone с notch/Dynamic Island | iPad или iPhone Plus/Max, увеличенный Dynamic Type |
